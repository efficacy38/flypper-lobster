# Generated Visual Assets

Visual assets were generated for this project with the built-in image generation tool.

- `bird.png`: side-view arcade bird sprite
- `pipe.png`: vertical obstacle texture
- `background.png`: portrait sky background
- `ground.png`: scrolling ground strip

# Font Asset

- `fonts/US101.ttf`: U.S. 101 font by Tom Oetken. The upstream font note states it is free for commercial and personal use.
